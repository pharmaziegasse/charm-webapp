## [](#quick-start)Quick start

- Clone the repository: `git clone https://github.com/pharmaziegasse/charm-webapp.git charm`
- Go to the new folder `cd charm`
- Run `npm install`
- Start with `npm start`

The demo site will now be accessible at [http://localhost:3000/](http://localhost:3000/).
